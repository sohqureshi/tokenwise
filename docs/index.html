TokenWise

Optimize JSON before sending it to LLMs. Reduce token usage, lower API costs, and improve performance instantly.

View on GitHub ❤️ Support Project

⚠️ The Problem

❌ Raw JSON increases token cost
❌ Extra structure wastes LLM budget
❌ Slower responses from large payloads

🚀 The Solution

TokenWise compresses and optimizes JSON before sending it to LLMs — reducing cost without losing meaning.

💻 CLI Usage

node cli/index.js optimize input.json

📉 Example

Input 
{
  "user": {
    "name": "John",
    "preferences": {
      "theme": "dark"
    }
  }
}
Output 
{"user":{"name":"John","preferences":{"theme":"dark"}}}
